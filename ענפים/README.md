# 🎵 דשבורד ניתוח נתוני ספוטיפיי

אפליקציה אינטראקטיבית לניתוח ויזואליזציה של נתוני Spotify, בנויה עם Streamlit.

## תכונות

- **העלאת קובץ CSV** – טעינה ותצוגה של נתוני Spotify
- **פילטרים** – שנה, ז'אנר, פופולריות, אומן, אורך שיר
- **סטטיסטיקות** – מספר שירים, אומנים, סך דקות מוזיקה
- **גרפים** – פופולריות לפי שנה, ז'אנרים פופולריים, אומנים מובילים
- **מטריצת קורלציה** – בין מאפיינים נומריים

## דרישות

- Python 3.8+
- streamlit
- pandas
- matplotlib

## התקנה

```bash
pip install streamlit pandas matplotlib
```

## הרצה

```bash
streamlit run test.py
```

האפליקציה תיפתח בדפדפן (בדרך כלל `http://localhost:8501`).

## שימוש

1. העלה קובץ CSV עם נתוני Spotify.
2. השתמש בפילטרים בסרגל הצד (שנה, ז'אנר, פופולריות, אומן, אורך שיר).
3. עיין בסטטיסטיקות, בגרפים ובמטריצת הקורלציה.

## מבנה קובץ CSV צפוי

האפליקציה תומכת בעמודות כמו:

| עמודה        | תיאור              |
|--------------|---------------------|
| `year`       | שנת השיר            |
| `genre`      | ז'אנר               |
| `artist`     | שם האומן            |
| `popularity` | ציון פופולריות      |
| `duration_ms`| אורך השיר (מילישניות) |

עמודות נוספות נומריות ייכללו במטריצת הקורלציה.
